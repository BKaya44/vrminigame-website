Introduction

Low poly art style

This game has a low poly art style. Low Poly is a style of design that translates an image into a polygon tessellation using tree sided polygons. I used this art style because I felt like using something which was easy to create 3D models in. For the next time I would like to use more cartoonish art style.

3D Models

All 3d models are made by me using Blender. When looking back at the first models I created and the ones that are in game, I really improved creating 3D models. In the future I want to use Blender for animating the objects I made.

VR

The unique thing about this game is that you can interact with all objects using a VR headset. While owning a VR headset is a must to play the game, it really gives you the feeling that you are in the game. Creating a VR game was really a challenge since there are not much tutorials on how to create certain things for VR. Most things are done by testing it and tweaking the things that are not working correctly. For future game development I would rather create my own engine for a better learning experience.
